# Sistema de colas - Frontend

## Requisitos previos

    Asegúrese de que Node.js (>= 10.13.0) esté instalado en su sistema operativo.

## Tecnologías

    Se utilizó ReactJs

## Instalación

    Git clone https://github.com/Levitaxl/sistema-de-colas.git
    cd sistema-de-colas
    npm i

## Ejecutar el código

    npm run dev
